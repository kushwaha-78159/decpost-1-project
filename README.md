# Disaster Guard AI 🚨

Ek modern disaster alert system jo user ki safety ko priority deta hai. Ye Next.js, pnpm, aur Groq LLM ka use karke real-time weather tracking aur disaster prediction karta hai.

## 🚀 Key Features

| Feature | Description |
|---------|-------------|
| **GPS Location** | [cite_start]Browser permission ke baad apne aap location detect karta hai[cite: 1, 2]. |
| **Weather Display** | [cite_start]Live Temperature, Humidity, aur Wind speed ka data dikhata hai[cite: 3]. |
| **Disaster Alert** | [cite_start]Flood, Cyclone, aur Heat wave jaise risks detect hone par alert bhejta hai[cite: 4]. |
| **SOS Button** | [cite_start]Ek click mein pre-saved contacts ko user ki current location SMS bhejta hai[cite: 5, 7]. |
| **Emergency Contacts** | [cite_start]Aap family aur friends ke numbers app mein save kar sakte ho[cite: 6]. |
| **Offline Mode** | [cite_start]Internet na hone par bhi last saved weather data display karta hai. |
| **SMS Alerts** | [cite_start]Risk detect hone par system automatically SMS notifications bhejta hai. |

## 🛠️ Tech Stack

- **Framework:** Next.js (App Router)
- **Package Manager:** pnpm
- **AI Engine:** Groq SDK (Llama-3 model)
- **Styling:** Tailwind CSS
- **Icons:** Lucide React
